## Platform Used

This project was developed using Wix as a No-Code website development platform.

---

## Project Description

The project is a simple e-commerce website designed to simulate an online shopping experience. The website includes product listings, store information, contact details, and cart interaction features.

---

## Features Implemented

* Homepage with welcome message
* Product page with products, images, prices, and descriptions
* About page describing the store mission
* Contact page with contact information and form
* Add-to-cart/cart interaction feature
* Responsive navigation and user-friendly design

---

## Website Pages

### Homepage

The homepage introduces the store and welcomes visitors.

### Product Page

The product page contains multiple products with:

* Product images
* Prices
* Descriptions

### About Page

This page explains the purpose and mission of the store.

### Contact Page

The contact page allows users to communicate through contact information and forms.

### Cart Interaction

Users can add products to cart for shopping simulation.
